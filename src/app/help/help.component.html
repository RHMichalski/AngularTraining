Cześć

Witam w przykładowej aplikacji stworzonej w trakcie nauki Angulara.

Aby w pełni wykorzystać możliwości, należy uruchomić Fake RestApi na swoim komputerze.
Aby to zrobić wystarczy:

  1. Zainstalować JSON server komendą: "npm install -g json-server"
  2. Pobrać plik db.json z repozytorium GitHub tego projektu.
  3. W konsoli Powershell ustawić odpowiedną ścieżkę do pliku db.json
  4. Uruchomić komendę: "json-server --watch db.json"
